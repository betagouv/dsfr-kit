# DSFR Components Overview

![Components Reference](/Users/luis/.gemini/antigravity/brain/2f480890-9099-4b2f-8249-e595a9baba11/uploaded_image_1765701501663.png)

This document lists the components available in the DSFR, categorized by function. We will document them progressively.

## Structure & Layout
- **En-tête (Header)**: The official Marianne header.
- **Pied de page (Footer)**: The official footer with mandatory links.
- **Bloc Marque (Logo)**: The official "République Française" block.

## Navigation
- **Fil d'Ariane (Breadcrumb)**: Path hierarchy.
- **Navigation principale**: Main menu bar.
- **Navigation latérale (Side Menu)**: Vertical navigation.
- **Onglets (Tabs)**: Content switching.
- **Pagination**: Sequence of pages.
- **Liens d'évitement (Skiplinks)**: Accessibility shortcuts.

## Actions & Forms
- **Bouton (Button)**: Primary user triggers.
- **Champs de saisie (Inputs)**: Text, dates, etc.
- **Cases à cocher (Checkboxes)**: Multiple selection.
- **Boutons radio (Radios)**: Single selection.
- **Sélecteur (Select)**: Dropdown options.
- **Barre de recherche (Search Bar)**.

## Content & Feedback
- **Accordéon (Accordion)**: Collapsible sections.
- **Badge**: Status indicators (colors, small text).
- **Tag**: Keywords or categories (can be clickable).
- **Carte (Card)**: Content summary with link.
- **Tuile (Tile)**: High-level navigation unit.
- **Citation (Quote)**: Highlighted text.
- **Mise en avant (Callout)**: Important information.
- **Alerte (Alert)**: Feedback messages (Success, Error, Info, Warning).
- **Modale (Modal)**: Overlay dialogs.
- **Tableau (Table)**: Data presentation.

## Font Families

### 1. Marianne (Primary)
- **Type**: Sans-serif.
- **Role**: Identity of the State. Standard for all text (Titles, Body, Interface).
- **Weights**:
  - **Bold (700)**: Emphasis, Titles.
  - **Medium (500)**: Emphasized body text.
  - **Regular (400)**: Standard body text.

### 2. Spectral (Secondary)
- **Type**: Serif.
- **Role**: Styling, Quotes, Editorials.
- **Weight**: **Regular (400)** only.
- **Usage**: Use sparingly to create contrast.

### Headings & Display
| Style | Usage |
| :--- | :--- |
| **Display** | Exceptional titles, very large impact (Landing pages). |
| **H1** | Page Title. Unique per page. |
| **H2** | Major section heading. |
| **H3** | Subsection heading. |
| **H4** | Minor subsection heading. |
| **H5** | Local group heading. |
| **H6** | Smallest heading level. |

### Text & Details
| Style | Role |
| :--- | :--- |
| **Lead** (Chapeau) | Introduction text, larger than body. |
| **Body** (Corps) | Standard text (Size `md`, `sm`, `xs` available). Default is 16px (1rem). |
| **Detail** | Legals, footnotes, captions (Small text). |
| **Alt** (Alternative) | Variants using bold/italic for specific contexts. |

## Usage Rules
1.  **Strict Hierarchy**: Always specific heading levels (`h1` -> `h6`) for accessibility.
2.  **Responsiveness**: Text sizes adjust automatically. Do not force fixed pixel sizes.
3.  **Line Height**: Standardized for readability.
4.  **Alternation**: Use Bold/Regular contrasts to create hierarchy within text blocks.
5.  **Alignment**: Left-aligned by default (reading direction). Justified text is generally avoided for accessibility.
